Helpful Resources

Published 7 December 2022 • Created by Xiomy "Cece" Sam.

A list of helpful resources I have encountered, including some for those who want to make a website of their own like this one!

Books | Websites | Videos


Books

#1: Learning Web Design

By Jennifer Nierderst Robbins

ISBN: 978-1-491-96020-2

Publisher: O'Reilly Media; 5th edition (June 19, 2018)

Edition: 5th

Link: Learning Web Design

I jokingly call this textbook “the Bible” whenever I have it on me or talk about it. This book is so vital that this page wouldn’t have been made without it! This book makes web design so much easier for a beginner like me/ Trust me when I say that this one also treats you like a person versus others that make you hit the ground with a horrific “Splat!” noise. I’m almost considering buying it for myself, considering how my major might have me expanding in the field of web design. This book has been nothing but helpful almost every time I used it! I highly recommend it and it’s learning materials!


#2 - Graphic Design Solutions

By Robin Landa

ISBN: 978-1-337-55405-3

Publisher: Cengage Learning

Edition: 6th

Link: Graphic Design Solutions (Amazon)

Now this is if you’re genuinely pursuing more creative and expressive routes. Web design and overall artistic design go hand-in-hand sometimes! Imagine spending hours creating a functional website, only for everything to show up on a white background with varying sizes of a black Times New Roman font. Alternatively, imagine you have to create a banner for your profile but don’t know where to start. This book will help with that! The examples are lovely, the book didn’t bore me to death with glossary terms and the examples shine as bright as the sun sometimes. Only downside is that the formatting can be a little unconventional at times, but I think that adds to the charm!


#3: Introduction to Java Programming and Data Structures, Comprehensive Version

By Y. Daniel Liang

ISBN: 978-0-136-52023-8

Publisher: Pearson

Edition: 12th

Link:

I have this book saved in my Amazon cart! Now granted, this is the heaviest book at almost 4 lbs but!! It is nothing but helpful. To some, they consider it “the Bible” of Javascript coding. I also agree with those people! I only explored about eight chapters of this book during my introductory course but I was using my money’s worth with those eight chapters. I would honestly like to rent this book again and see what else I can do with its definitions and examples! The only downside (outside of price) is that the font can be small and like I mentioned before, the weight of this book is out there.

Websites

#1: w3c Schools

Link: w3schools

This little nifty website has aided me in some small ways while making this website, so I figured I would share.!They cover so many realms when ti comes to coding that it’s no surprise when they almost always show up for my search results! Apparently they can help you get certified as well, which doesn’t hurt!


#2: Color Picker

Link: HTML Color Picker

Absolutely wonderful for finding a color palette. Incredibly ease to use, lets you to lock colors to make your own palette, allows you to download your palettes and even describes what the different color options are. Very nice for beginners!


#3: Color Palettes

Link: Adobe Color

Provides mostly the same content as the previous website, just provided by Adobe. It’s more about giving colors than anything, along with deep dives behind the color values chosen.


#4: Google Fonts

Link: Google Fonts

So! You like fonts/typefaces but don’t know where to go. One stop shop right here with Google Fonts! (Hehe! My title is a Google font.) It’s like the Adobe Fonts website except free! Might not get the exact same fonts, but it’s still incredibly reliable. Definitely would recommend to those who want to add more flavor to their websites.

TOP


Videos

#1: Intro to Java Programming - Course for Absolute Beginners

Link: Youtube

This one is more like a personal bookmark to me. Having to switch between JavaScript and HTML was interesting to say the least. Now I will be honest, I only spent an hour working with this video but! At no point did I feel smaller, dumber or awful when following this man’s tutorial. It has shining reviews in the comments about how insightful he is compared to actual JavaScript classes. I would recommend this video for someone just starting out!
